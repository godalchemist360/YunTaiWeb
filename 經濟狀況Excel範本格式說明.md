# 經濟狀況 Excel 匯入/匯出格式說明

## 檔案結構

使用**單一工作表**，包含四個區塊：收入、支出、資產、負債

## 格式規則說明

### 1. 收入區塊
- **標題行**：`【收入】`
- **欄位**：項目名稱 | 金額
- **說明**：
  - 第一列為標題（【收入】）
  - 第二列為欄位標題（項目名稱、金額）
  - 從第三列開始為資料
  - 項目名稱可以是預設項目（主業收入、副業收入）或自訂項目
  - 金額為數字，單位為元

### 2. 支出區塊
- **標題行**：`【支出】`
- **欄位**：分類 | 項目 | 金額
- **說明**：
  - 第一列為標題（【支出】）
  - 第二列為欄位標題（分類、項目、金額）
  - 從第三列開始為資料
  - **分類**：必須是以下之一
    - 必要性
    - 雜支
    - 娛樂
    - 理財投資
    - 人情
    - 其他
  - **項目**：
    - 如果是「其他」分類，項目名稱可自訂
    - 其他分類必須使用對應的子分類項目（見下方對照表）
  - **金額**：數字，單位為元
  - **空行**：支出區塊結束後需有一個空行

### 3. 資產區塊
- **標題行**：`【資產】`
- **欄位**：項目名稱 | 金額
- **說明**：
  - 第一列為標題（【資產】）
  - 第二列為欄位標題（項目名稱、金額）
  - 從第三列開始為資料
  - 項目名稱可以是預設項目（不動產、現金、股票ETF、基金、保險）或自訂項目
  - 金額為數字，單位為元

### 4. 負債區塊
- **標題行**：`【負債】`
- **欄位**：項目名稱 | 金額
- **說明**：
  - 第一列為標題（【負債】）
  - 第二列為欄位標題（項目名稱、金額）
  - 從第三列開始為資料
  - 項目名稱可以是預設項目（房貸、車貸、信貸、卡循、學貸、融資分期）或自訂項目
  - 金額為數字，單位為元

## 支出分類對照表

| 分類 | 可用的項目（子分類） |
|------|---------------------|
| 必要性 | 生活伙食、租金 |
| 雜支 | 交通、電話 |
| 娛樂 | 訂閱費、治裝費、興趣愛好 |
| 理財投資 | 定期定額、貸款還款 |
| 人情 | 孝親費 |
| 其他 | 可自訂任何項目名稱 |

---

## 注意事項

1. **標題必須完全一致**：`【收入】`、`【支出】`、`【資產】`、`【負債】`（包含全形括號）
2. **區塊順序**：必須按照 收入 → 支出 → 資產 → 負債 的順序
3. **空行**：每個區塊之間必須至少有一個空行分隔
4. **金額格式**：
   - 只接受數字
   - 不支援千分位符號（如：1,000）
   - 不支援貨幣符號（如：$、NT$）
   - 小數點後最多兩位
5. **空值處理**：
   - 金額為 0 或空白的項目會被忽略
   - 項目名稱為空白的行會被忽略
6. **匯入驗證**：
   - 支出分類必須符合對照表
   - 支出項目必須符合對應分類的子分類（「其他」分類除外）
   - 金額必須為有效數字
